| **半导体制冷**      | 电流：1A ~ 15A；电压：5V 或 12V  | -                                                 | 核心元件，配置多样  |
| **充电宝 (供电)**   | 常见：5V / 2A (100元内)；12V少见 | 市场存在虚标情况                                          | 常见，价格低廉    |
| **水冷 (液体循环)**  | 管道遍布全身，加冰水，有循环泵          | 制冷快，结构简单                                          | 有此类产品      |

| **相变材料 (PCM)** | 冰块/PCM材料辅助，需预先冷冻         | 无电子设备，可持续工作约4小时                                   | -          |
| **石墨烯传导**      | 利用石墨烯材料传导热量              | -                                                 | 市面未见此产品    |
